# Cognetix Student Management System 📊

This is a console-based Student Management System developed as part of the Cognetix Python Developer Internship.

## 🚀 Features
- Add new student records
- View all students
- Search student by ID
- Delete student records
- Data stored using CSV file

## 🛠️ Technologies Used
- Python
- CSV File Handling

## ▶️ How to Run
1. Open terminal
2. Run:
   python student_manager.py

## 📂 Project Structure
- student_manager.py → Main program
- students.csv → Stores student data
- README.md → Project documentation

## 📸 Sample Output
- Student added successfully
- Student search result
- Student deleted successfully

## 🎯 Learning Outcome
- File handling using CSV
- CRUD operations
- Input validation
- Python functions and logic
